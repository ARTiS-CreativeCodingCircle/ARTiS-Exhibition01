# 操作方法
キーボードとマウスを用いて操作します。  

<a id="chapter1-1"></a>

## キーボード
キーボードのWASDキーを使います。  
- Wキー：カメラの方向から前に進む
- Aキー：カメラの方向から左に進む
- Sキー：カメラの方向から後ろに進む
- Dキー：カメラの方向から右に進む
<br/>
また、**左Shiftキー**を押しながらキーを押すと移動速度が上昇します。

<a id="chapter1-2"></a>

## マウス
左クリックしながらマウスを動かすとカメラが動きます。  
また、動画作品では作品を左クリックすることで再生と停止を行うことができます。  
(動画再生は動作が重くなる可能性があるため、基本は停止状態にしておくことを推奨します)
